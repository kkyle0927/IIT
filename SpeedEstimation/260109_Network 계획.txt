Spatiotemporal parameters (가장 직접적으로 gait speed를 설명)
Cadence (step frequency) / Step length / Stride length / Stride time / Stance duration / Swing duration

Joint kinematics (주로 sagittal plane, 예측력은 중간 이하)
Peak hip flexion angle / Peak hip extension angle / Peak knee flexion angle (loading response) / Peak knee flexion angle (swing) / Peak knee extension angle (terminal stance) / Peak ankle plantarflexion angle / Peak ankle dorsiflexion angle (mid-stance, swing)

Joint kinetics – moments (gait speed와 가장 강한 연관)
Peak hip flexion moment (stance) / Peak hip extension moment / Peak hip flexion moment (swing) / Peak knee flexion moment (loading response) / Peak knee extension moment (terminal stance) / Peak knee flexion moment (pre-swing) / Peak knee extension moment (swing) / Knee abduction moment (frontal plane, 특히 고속 보행 시 증가) / Peak ankle dorsiflexion moment / Peak ankle plantarflexion moment

Joint kinetics – powers (속도 민감도 매우 높음)
Peak hip power generation (loading response) / Peak hip power absorption / Peak hip power generation (pre-swing) / Peak knee power absorption (loading response) / Peak knee power generation (mid-stance) / Peak knee power absorption (pre-swing) / Peak knee power absorption (swing) / Peak ankle power generation (pre-swing) / Peak ankle power absorption

Ground reaction forces (GRF)
Vertical GRF first peak / Vertical GRF second peak / Braking force (anterior-posterior) / Propulsive force (anterior-posterior)

1순위: Cadence / Step length / Stride length
2순위: Hip, knee, ankle joint moments / Joint power (특히 ankle push-off, hip pre-swing power)
3순위: GRF (propulsive, braking, vertical peak)
4순위: Joint angles (kinematics, 보조적 설명 변수)
